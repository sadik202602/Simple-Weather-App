Simple Weather App ☁️
 
Hi!
This is a small Python project where I made a simple weather app using the OpenWeather API.

You can type a city name and the app will show:

• current temperature
• weather condition
• a GIF animation depending on the temperature

This project helped me practice working with APIs, Python, and a simple GUI.
Tools

• Python
• Tkinter
• Requests
• OpenWeather API

How to run

1. Install requests
pip install requests
2. Add your OpenWeather API key in the Python file.
3. Run the program
python weather_app.py
