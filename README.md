# 🌤️ Simple Weather App

This is my first Python project.  
It is a simple weather application that shows the temperature of a city using an API.


## Features
- Enter a city name
- Get temperature in Celsius
- Simple GUI using PyQt5


## Built With
- Python
- PyQt5
- Requests
- OpenWeather API

## How to Run

1. Install required libraries:
   
pip install PyQt5 requests
